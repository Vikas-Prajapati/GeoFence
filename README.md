# GeoFence
This is a Geofence based applicaton. Geofence is a virtual boundary set around a geographical area like market, school etc. This application was created keeping in mind the scenario of child's safety. The applicaton works like this, a geofenced area is created aroud the location like a school whenever the child device(or child) leaves or enters that area a message is triggered, sent to the mobile number specified in the Settings page of the appplication.
The apllication has made with the use of Firebase due to some reasons:
1. Firebase has helped to reduce the code to considerable extent.
2. Firebase has some add-on like authentication, realtime-database, that could be included in the application to.
